# Lista de Recados 2.0

Upgarede do projeto de lista de recados desenvolvido para finalização do módulo de front end II - Growdev

## Tecnologias utilizadas

- React
- React Router DOM
- Typescript
- Redux
- Redux Toolkit
- Redux Persist
- MUI - Material Design
